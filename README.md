## Getting Started

- Run this command in your OS terminal: `git clone https://github.com/notrexxx/microverse-ABB-enumerables.git` to get a copy of the project.


### Prerequisites

* On Windows:
Have Ruby installed
* Mac & Linux:
Latest version of Ruby installed

### Setup

* Clone the repository

### Usage

* On Windows:
Ruby enumerables.rb
* Mac & Linux:
$ Ruby enumerables.rb
